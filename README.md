# SushiGo
Sushi Go is a board game developed by Phil Walker-Harding and Gamewright. I have made translated the rules of the game into a digital format using Processing. I did this using an object-oriented approach. This game was developed chiefly during my senior year of high school, and I incorporated most of what I had learned about programming in java up to that point into this project in some form. I would now like to share my creation with other people. The game has achievements and a final boss, and tracks how much time it takes you to beat the game. My best is around 55 minutes, it will probably take a while for anyone else to finish it in one sitting without figuring out how the game works and developing a strategy for how to beat the game quickly. Right now there is no saving feature or any leaderboards, but these are features that I am interested in implementing in the future. So for now, enjoy all the Sushi you can!
